## Step 4 — The wizard (20:50 UTC)

`/wizard` route ships. Six steps, autosave per step:

1. **Welcome** — vaporwave-tinted hero (perspective grid + sunset gradient), copy beat only, no persistence.
2. **Profile** — name, current-role one-liner, exploring-text, location prefs (UK/EU/US/Remote toggles + notes). Debounced autosave to `user_profile`.
3. **Frames** — read-only summary of the six pre-seeded scale frames with inline rename on blur. "Edit on /frames" link out for deeper edits (description / new frames / kind changes).
4. **Weighing** — Must / Should / Could buttons per frame, maps the existing `frameWeights` `low|medium|high` schema 1:1 (Must = high, Should = medium, Could = low). Per-click autosave.
5. **Open-text thoughts** — three default prompts ("what's making this hard / what's a great outcome / what are you afraid you're optimising for"). Debounced autosave.
6. **The big moment** — kicks `completeWizard`: sets `wizardCompletedAt`, fans rescore across all companies via Next 15's `after()` so the user isn't blocked. Theatre window: 25-sec progress bar, cycling `firstScoring[]` quotes (rotates every 3.5s), grid + sunset backdrop. Auto-bounces home on complete.

Front door: `src/app/page.tsx` now redirects to `/wizard` when `wizardCompletedAt` is null. `/wizard` redirects back home when set, except in `?replay=1` mode (used by the future /about replay-onboarding button).
