- Verified routes: `/`, `/companies/anthropic`, `/compare`, `/frames` return 200 with cookie; without cookie 307 → `/login`.

**Why:**
- Drizzle's `onConflictDoUpdate` makes the seed reentrant; can re-run safely as data improves.
- Frame scores via inline radio-button-style buttons (not a slider) because clicking a number is faster than tweaking a handle, especially on the comparison view.
- Fit notes call Claude server-side (server action) so the API key never reaches the browser.
- Password gate beats no auth and is removable in ~5 lines when Resend lands.

**Decisions:**
- Chose Sonnet (`claude-sonnet-4-5`) for fit-note generation — faster and cheaper than Opus, sufficient quality for 4–6 sentence notes.
- Fit-note generation is an explicit user action (button-click), not auto. Reduces accidental token spend and gives the user agency.
- Middleware matcher excludes `/api`, `_next/static`, `_next/image`, `favicon.ico` so the gate doesn't block assets or future webhooks.
- Did not implement frame *editing* UI yet — frames come from seed for v0.1; a `/frames/new` page is queued for v0.2.
- Did not pull live job feeds yet — the seed includes hand-curated current openings (Anthropic, OpenAI, ElevenLabs, Wayve). Greenhouse/Lever/Ashby clients come in v0.3.

## 16:48 UTC — Design system absorbed from europe2031.ai

**Did:**
- Read user reference: https://europe2031.ai/. Extracted palette + type personality from the HTML directly.
- Wrote `docs/DESIGN.md` codifying the system: warm off-white surfaces, ink + slate text, single muted-blue accent, warm clay-red for signal, hairline rules only.
- Typography: **Fraunces** (serif, human words), **Inter** (small UI), **Geist Mono** (data + eyebrows). Two type personalities is what does most of the design work.
- Wired tokens into Tailwind v4 `@theme` and rewrote `globals.css`.
- Updated `layout.tsx` to load the three font families with CSS variables.
- Rewrote `/` to use the new system: eyebrow + serif headline + restrained meta strip. No shadows, no gradients, no SaaS-y rounding.
- Build is clean.

**Why:** User asked for the aesthetic to feel like europe2031.ai — editorial, thoughtful, grown-up. That's a much better fit for a policy-thinking tool than a stock dashboard look. It also reads as commissioned-quality work, which serves the surprise discipline.

**Decisions:**
- Picked Fraunces over alternatives (Newsreader, Söhne) because it's free, variable, and has optical sizing built in — fits the editorial brief without licensing friction.
- No dark mode in v0. The warm off-white is the whole aesthetic.
- Cat presence kept to two locations only: the wordmark and the future "lobbycat says" callout. Discipline.

**Next:** Drizzle install + schema + first migration to Neon.
